Connect Odoo client and 3CX

Click here for the demo video

Prerequisites

1. End user must use a computer running Windows. This module does not support other platforms.

2. 3CXPhone for Windows (soft phone)is installed and working.

3. An user must use Odoo and 3CX soft phone in the same computer.

4. PRO license of 3CX phone system. Free trial license does NOT work. Learn more

How to install

There are 2 folders in this git repository: threecx and NodoCRMPlugin.
  • threecx is an Odoo module.
  • NodoCRMPlugin is a plugin for '3CX Phone for Windows'.

Installation steps:

1. Install threecx for Odoo. This should be straight forward.

2. Copy NodoCRMPlugin/NodoCRMPlugin.dll into 3CX Phone for Windows installation folder. Typically, it is: 

C:\ProgramData\3CXPhone for Windows\PhoneApp

3. Still in the above PhoneApp folder, open '3CXWin8Phone.user.config', add NodoCRMPlugin into value of the line 'CRMPlugin'. Full content of this file should be like this:

<?xml version="1.0" encoding="utf-8"?>
<appSettings>
  <add key="version" value="3.0.0" />
  <add key="ClientSettingsProvider.ServiceUri" value="" />
  <add key="CRMPlugin" value="CallTriggerCmdPlugin,DotNetScriptPlugin,NodoCRMPlugin" />
  <add key="MakeCallTimeout" value="10000" />
  <add key="MakeCallTimeoutCTIUnsupported" value="200000" />
  <add key="EnableVCELogging" value="true" />
  <add key="MaxLogFileSizeMb" value="5" />
  <add key="RequestTimeout" value="60" />
  <add key="MaxUploadSizeKb" value="2048" />
</appSettings>

4. Restart 3CX Phone for Windows and refresh Odoo web page. Odoo client now should be connected to 3CX soft phone.
